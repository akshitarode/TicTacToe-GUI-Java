# TicTacToe GUI - Java Swing

A 2-player TicTacToe game with graphical interface built using Java Swing.

## Features
- GUI with 3x3 Grid Buttons
- Turn indicator: Player X / Player O  
- Win and Draw detection
- Reset button to play again

## Tech Stack
- Java
- Java Swing - JFrame, JButton, ActionListener
- OOP Concepts

## How to Run
1. Open terminal in project folder
2. Compile: javac *.java
3. Run: java TicTacToe
